WatchNow is a project for Microsoft Engage 2022 and is made for the demonstration of algorithms.
The project is made between the start of Engage mentorship program (May 05 2022) and final submission date (May 27 2022) As the name suggest, it suggest content based on what you liked and it offers what will you like. The data used in making of the WatchNow comes from mainly 2 places.
  • Grouplens movielens Dataset
  • The Movie Database API's
WatchNow has data collection of 7M+ users and 1K+ movies. Only user rating data is being taken up.
